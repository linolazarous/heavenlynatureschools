# POST-DEPLOYMENT CHECKLIST

Use this checklist after deploying your website to ensure everything is configured correctly.

## ✅ IMMEDIATE POST-DEPLOYMENT TASKS

### 1. Domain & SSL Setup
- [ ] Domain purchased and connected to Netlify
- [ ] DNS records properly configured (A and CNAME)
- [ ] SSL certificate provisioned and active
- [ ] www redirect working correctly
- [ ] HTTPS is enforced

### 2. Netlify Identity Configuration
- [ ] Netlify Identity enabled in dashboard
- [ ] Registration set to "Invite only"
- [ ] Admin users invited via email
- [ ] Admin users can successfully log in
- [ ] Git Gateway enabled (for future features)

### 3. Netlify Forms Setup
- [ ] Contact form submissions working
- [ ] Email notifications configured
- [ ] Test submission received at info@heavenlynatureschools.com
- [ ] Spam protection enabled

### 4. Email Configuration (Zoho Mail)
- [ ] Zoho Mail account created
- [ ] MX records added to domain DNS
- [ ] SPF record configured
- [ ] DKIM record configured
- [ ] info@heavenlynatureschools.com is functional
- [ ] Test email sent and received

## ✅ CONTENT POPULATION

### 5. Initial Content Creation
- [ ] Create first blog post about school opening
- [ ] Add upcoming events (if any)
- [ ] Test blog post appears on public site
- [ ] Test event appears on events page

- [ ] Can create new blog posts

- [ ] Can delete blog posts
- [ ] Can create new events

- [ ] Can delete events
- [ ] Can view contact form submissions
